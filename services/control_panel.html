HTTPD/Dispatcher controls

  

  

    

      Control UI listens on CONTROL_PORT (default 59173), not through the dispatcher. Dispatcher vhost is port 80.
    

    
Dispatcher module version

    

      Versions match Adobe-published Linux tarballs. The module is downloaded only from this page (nothing proprietary is baked into the image).
